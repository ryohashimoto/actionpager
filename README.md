# Action Pager

Simple and lightweight paginator for collections (Array, ActiveRecord::Relation etc.)

## Installation

Add this line to your application's Gemfile:

```ruby
gem "actionpager"
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install actionpager
